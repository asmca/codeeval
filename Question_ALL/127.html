  • Code Plagiarism

    Challenge Description:

    We propose you to solve a very interesting challenge - to find plagiarism in one source code against the code of another program. This problem is similar to finding plagiarism between two texts, but in case of source code we know the syntax of a language and we can be more accurate in finding renamed data structures and other cheating.

    At the moment for this task we are not providing any ranking points. But we will give a special badge to those developers who will send us a good working solution.

    To make things even more interesting we have chosen two programming languages: Python and Go. You will have two pairs of source code to compare. The first pair of code is written with python. The second pair of code is written with Go. You need to find duplicate in pairs, not between them. Thus you won't need to compare Python code and code written with Go against each other.

    Our test cases for both languages are: 

    1) The same code
2) Changed names of variables
3) One is a half of another
4) Removed documentation strings and comment
5) Shuffled functions
6) Small part of code is inside of another code
7) Two approaches to the same problem
8) Different code
9) "Hello World"

    Please feel free to write us at support at codeeval.com if you think that you can add more test cases which are suitable for this task. Also feel free to write to us if you need to install some library for the language you are using for successful completion of the challenge.

    Input sample:

    Your program should accept as its first argument a path to a filename. The input file contains 18 sets of test cases (9 for Python and 9 for Go). You can check the input file on GitHub Each test case is separated by five asterisks. The first line of each test case is it's title e.g. "<<< Python The same code". Two code samples inside of each test case are separated by five equal signs. E.g. 

    <<< Python The same code   # first test case

def quickSort(alist):
    ... new code

=====

def quickSort(alist):
    ... existing code

*****
<<< Go The same code  # second test case

package main
    ... new code

=====

package main
    ... existing code

*****
<<< Python Changed names of variables  # Third test case
            etc.

    Output sample:

    For each test case produce a single line of output containing the percentage of plagiarism comparing new code against existing one. E.g. 

    100
95
0
